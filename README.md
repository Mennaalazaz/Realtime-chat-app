# Chat App — Django Real-Time Messaging System
## Overview
This is a real-time chat application built using Django. The project allows users to join chat rooms and exchange messages through a simple and clean user interface.

The application demonstrates:

  - Django backend development

  - Database integration using SQLite

  - Template rendering

  - Static file management
    
## Features

- 🏠 Create or join chat rooms

- 💬 Real-time messaging

- 📦 Database-backed message storage

- 🎨 Clean and responsive UI

## Installation & Setup
### 1. Clone Repository
```
git clone https://github.com/Mennaalazaz/Realtime-chat-app.git
cd chatapp
```

### 2. Install Dependencies
```
pip install django
```

### 3. Run Migrations
```
python manage.py migrate
```

### 4. Run Server
```
python manage.py runserver
```

#### then open
```
http://127.0.0.1:8000
```
